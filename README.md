The **Library Management System** is a web-based application developed using **Java Servlets, JDBC, HTML, and Oracle Database**.
This project allows an administrator to manage book records in a library by adding new books and viewing existing book details.

It demonstrates the implementation of **MVC architecture** using Java technologies and database connectivity.


* Add a new book to the library
* View/search book details using ISBN
* Store and retrieve data from Oracle database
* Display book and author information
* Input validation and error handling
* Simple and user-friendly interface

 Technologies Used

* Java (Servlets & JDBC)
* HTML
* Apache Tomcat Server
* Oracle Database (SQL)
* Eclipse IDE
* MVC Architecture

---

Modules
Add Book

* Enter ISBN, book title, book type, author, and cost
* Stores book details in database

View Book

* Search book using ISBN
* Displays book details including author information

 Database Tables

Author Table

* Author Code (Primary Key)
* Author Name
* Contact Number

 Book Table

* ISBN (Primary Key)
* Book Title
* Book Type
* Author Code (Foreign Key)
* Book Cost
